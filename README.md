Hercios es un proyecto experimental que propone un cambio: utilizar el sonido en lugar de la imagen para representar la identidad. El objetivo es que la identidad se base en la personalidad, no en la apariencia. El sistema de "Symphony" se presenta como una plataforma digital y funciona de la siguiente manera:
- Test de Identidad: el usuario realiza un test que analiza sus rasgos de carácter principales.
- Rasgos a notas: cada rasgo de la personalidad se traduce en una nota musical (frecuencia de sonido concreta).
- Acorde personal: estas notas se combinan para crear un acorde específico y único para el usuario, su identidad sonora.
- Personalidad a instrumento: en función del tipo general de personalidad, se asigna al usuario un instrumento (timbre). <br/>

De esta manera, se genera una identidad sonora clara para cada persona. Es una identidad basada en los rasgos de carácter reales e internos, dejando de lado las imágenes y las apariencias.

Puedes probar Hercios [aquí](https://samuelquiros-design.github.io/hercios/) para ver más detalles.
